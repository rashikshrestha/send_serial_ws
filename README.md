## Follow folling steps:

```
git clone https://github.com/rashikshrestha/send_serial_ws.git
cd send_serial_ws
catkin_make
sudo ./init_send_serial.sh
```

## To send serial data

### In terminal 1:
To send serial data
```
send_serial
```

### In terminal 2
To send command velocity
```
keyboard
```

To move around type(inside terminal 2): 
uio
jkl
m,.

### In terminal 3
To visualize the command velocity
```
show_cmd_vel
```
